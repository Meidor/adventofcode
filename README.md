# Advent of Code 2021

## Run

```sh
cargo run
cargo run day01
cargo run all
```

## Run tests

```sh
cargo test
cargo test day01
```

## Run benchmarks

```sh
cargo install cargo-criterion
cargo criterion
cargo criterion day01
```
